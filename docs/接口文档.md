# 接口文档

本文档用于记录官网后台系统与 Portal 前台接口。每新增、修改、删除一个接口，必须同步更新本文档。

---

## 维护规则

* **接口变更必须更新**：新增接口、修改接口路径、修改请求参数、修改响应字段、修改错误码、删除接口时，必须同步更新本文档。
* **必须标注实现位置**：每个接口条目必须写明 Controller 文件路径和方法名，便于快速定位代码。
* **必须区分接口类型**：后台接口归入 Admin 接口，前台公开接口归入 Portal 接口。
* **必须记录鉴权要求**：每个接口必须说明是否需要登录、需要什么权限、是否可匿名访问。
* **必须记录数据可见性**：Portal 接口必须说明是否过滤隐藏和逻辑删除数据；草稿、审批、定时发布等状态仅在PRD明确引入后记录。
* **必须记录错误码**：接口可能返回的业务错误码必须写入对应条目，错误码按 `docs/错误码说明.md` 中的数字分段规则设置。
* **文档先于交付**：接口代码完成但本文档未更新，视为接口未完成。

## 一、Admin 后台接口

### 后台登录

* **接口类型**：Admin
* **请求方法**：POST
* **接口路径**：`/admin/api/auth/login`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/system/controller/AdminAuthController.java#login`
* **鉴权要求**：无需登录，必须携带有效 CSRF Token。
* **功能说明**：校验后台账号密码，建立 Session 登录态并返回当前用户基础身份信息。
* **请求参数**：
    * `username`：string，必填，后台登录账号，最长 64 字符。
    * `password`：string，必填，后台登录密码，最长 128 字符。
* **响应字段**：
    * `userId`：long，后台用户 ID。
    * `username`：string，后台登录账号。
    * `displayName`：string，后台显示名称。
    * `roleCode`：string，后台角色编码，当前仅支持 `ADMINISTRATOR`。
* **错误码**：
    * `10001`：请求体缺少用户名或密码，或参数长度不合法。
    * `20003`：用户名或密码错误。
    * `20004`：账号已禁用，禁止登录。
    * `20005`：缺少 CSRF Token 或 Token 无效。
* **备注**：登录成功后写入 HttpSession；默认使用 Cookie Session 同域部署模式。

* **备注**：接口会记录退出登录安全审计日志占位信息。

### 获取当前登录用户

* **接口类型**：Admin
* **请求方法**：GET
* **接口路径**：`/admin/api/auth/me`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/system/controller/AdminAuthController.java#currentUser`
* **鉴权要求**：需登录；需角色：管理员。
* **功能说明**：返回当前后台登录用户的基础身份信息，供前端初始化登录态和权限展示。
* **请求参数**：无。
* **响应字段**：
    * `userId`：long，后台用户 ID。
    * `username`：string，后台登录账号。
    * `displayName`：string，后台显示名称。
    * `roleCode`：string，后台角色编码。
* **错误码**：
    * `20001`：未登录访问。
* **备注**：仅返回当前登录用户基础信息，不返回密码哈希、状态等敏感字段。

### 获取 CSRF Token

* **接口类型**：Admin
* **请求方法**：GET
* **接口路径**：`/admin/api/auth/csrf`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/system/controller/AdminAuthController.java#csrf`
* **鉴权要求**：无需登录。
* **功能说明**：为后台前端建立首个会话并返回当前会话的 CSRF Token 信息。
* **请求参数**：无。
* **响应字段**：
    * `token`：string，当前会话可用的 CSRF Token。
    * `headerName`：string，前端提交 CSRF Token 的 Header 名。
    * `parameterName`：string，前端提交 CSRF Token 的参数名。
* **错误码**：
    * `80000`：系统内部异常。
* **备注**：同时通过 Cookie 下发 `XSRF-TOKEN`，前端首次登录前应先调用该接口。

### 上传公共图片资源

* **接口类型**：Admin
* **请求方法**：POST
* **接口路径**：`/admin/api/media/assets`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/media/controller/AdminMediaAssetController.java#uploadImage`
* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：上传站点配置等模块复用的公共图片资源，返回 `mediaId` 和公开访问 URL。
* **请求参数**：
    * `file`：multipart file，必填，仅支持图片文件，默认最大 2MB。
* **响应字段**：
    * `mediaId`：long，媒体资源 ID。
    * `url`：string，公开访问 URL。
    * `contentType`：string，媒体 MIME 类型。
    * `size`：long，文件大小，单位字节。
* **错误码**：
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `20005`：缺少 CSRF Token 或 Token 无效。
    * `60001`：文件为空、扩展名/MIME/内容签名不合法或文件过大。
* **备注**：文件保存到应用外部本地目录，并记录媒体元数据；不允许直接上传可执行脚本。

### 获取导航菜单

* **接口类型**：Portal
* **请求方法**：GET
* **接口路径**：`/portal/api/site/navigation`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/PortalNavigationMenuController.java#getMenuTree`
* **鉴权要求**：无需登录，可匿名访问。
* **功能说明**：返回官网前台公开可见的顶部导航树。
* **请求参数**：无。
* **响应字段**：
    * `id`：long，菜单 ID。
    * `menuName`：string，菜单名称。
    * `targetType`：string，跳转类型。
    * `routePath`：string，内部路由，仅 `INTERNAL_ROUTE` 时返回。
    * `anchorCode`：string，页面锚点，仅 `PAGE_ANCHOR` 时返回。
    * `externalUrl`：string，外部链接，仅 `EXTERNAL_LINK` 时返回。
    * `openInNewTab`：boolean，外部链接是否新窗口打开。
    * `children`：array，二级菜单列表。
* **错误码**：
    * `80000`：系统内部异常。
* **备注**：只返回 `visible=1` 且 `deleted_marker=0` 的菜单；若分组型一级菜单没有任何可见二级菜单，则该父菜单不返回；默认走 Redis 缓存，缓存键为 `official:portal:navigation`。

### 获取首页核心数据指标卡片

* **接口类型**：Portal
* **请求方法**：GET
* **接口路径**：`/portal/api/site/home-metrics`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/PortalHomeMetricCardController.java#getMetricCards`
* **鉴权要求**：无需登录，可匿名访问。
* **功能说明**：返回官网前台公开可见的首页核心数据指标卡片列表。
* **请求参数**：无。
* **响应字段**：
    * `value`：string，数值核心区，纯数字字符串。
    * `unit`：string，单位或后缀；未配置时返回空字符串。
    * `description`：string，底部描述文案。
* **错误码**：
    * `80000`：系统内部异常。
* **备注**：只返回 `visible=1` 且 `deleted_marker=0` 的卡片，按 `sortOrder ASC, id ASC` 排序；`unit` 未配置时统一返回空字符串；默认走 Redis 缓存，缓存键为 `official:portal:home:metrics`。
