# 接口文档

本文档用于记录官网后台系统与 Portal 前台接口。每新增、修改、删除一个接口，必须同步更新本文档。

---

## 功能点索引

接口按以下功能点组织：

### 1. 全局配置与基础模块
- **站点基础配置**：获取站点配置（Admin/Portal）、更新站点配置
- **导航菜单管理**：获取菜单树、新增菜单、更新菜单、更新显示状态、删除菜单、重排菜单（Admin）、获取前台菜单（Portal）

### 2. 首页模块
- **首页首屏 Banner**：获取 Banner（Admin/Portal）、更新 Banner
- **首页核心数据指标**：获取指标列表、新增指标、编辑指标、更新显示状态、删除指标、重排指标（Admin）、获取前台指标（Portal）

### 3. 企业实力模块
- **企业实力核心指标**：查询指标列表、新增指标、编辑指标、删除指标、批量排序（Admin）、获取前台指标（Portal）

### 4. 客户与合作模块
- **客户 Logo 管理**：分页查询、新增、更新、删除、批量排序（Admin）、获取前台列表（Portal）
- **荣誉标签管理**：获取列表、新增、更新、删除、批量排序（Admin）、获取前台列表（Portal）

### 5. 关于我们与发展历程模块
- **合作高校管理**：获取列表、新增、更新、删除、批量排序（Admin）、获取前台列表（Portal）
- **重点研发方向管理**：获取列表、新增、更新、删除、批量排序（Admin）、获取前台列表（Portal）

### 6. AI 战略模块
- **AI 战略卡片管理**：分页查询、新增、更新、删除、批量排序（Admin）、获取前台列表（Portal）

### 7. 核心能力底座模块
- **底座分类与子项管理**：获取完整分类与子项树状列表、创建/编辑分类、创建/编辑子项、批量排序（Admin）、获取前台展示的能力底座分类列表（Portal）

### 8. 产品矩阵模块
- **产品信息管理**：分页查询产品列表、新增产品、编辑产品、删除产品、批量更新产品排序（Admin）、获取前台产品矩阵列表（Portal）
- **标杆案例管理**：分页查询案例列表、新增案例、编辑案例、删除案例、批量更新案例排序（Admin）、获取前台案例列表（Portal）
- **标杆案例管理**：分页查询案例列表、新增案例、编辑案例、删除案例、批量更新案例排序（Admin）、获取前台案例列表（Portal）

### 9. 媒体与系统
- **媒体资源管理**：上传媒体资源（Admin）
- **认证与授权**：登录、登出、获取当前用户、获取 CSRF Token（Admin）
- **标杆案例管理**：案例列表、创建、编辑、删除、重排（Admin）、前台案例列表（Portal）
- **标杆案例管理**：案例列表、创建、编辑、删除、重排（Admin）、前台案例列表（Portal）

## 维护规则

* **接口变更必须更新**：新增接口、修改接口路径、修改请求参数、修改响应字段、修改错误码、删除接口时，必须同步更新本文档。
* **必须标注实现位置**：每个接口条目必须写明 Controller 文件路径和方法名，便于快速定位代码。
* **必须区分接口类型**：后台接口归入 Admin 接口，前台公开接口归入 Portal 接口。
* **必须记录鉴权要求**：每个接口必须说明是否需要登录、需要什么权限、是否可匿名访问。
* **必须记录数据可见性**：Portal 接口必须说明是否过滤隐藏和逻辑删除数据；草稿、审批、定时发布等状态仅在PRD明确引入后记录。
* **必须记录错误码**：接口可能返回的业务错误码必须写入对应条目，错误码按 `docs/错误码说明.md` 中的数字分段规则设置。
* **文档先于交付**：接口代码完成但本文档未更新，视为接口未完成。

### 1. 全局配置与基础模块

#### 1.1 站点基础配置

### 获取站点基础配置

* **接口类型**：Admin
* **请求方法**：GET
* **接口路径**：`/admin/api/site/config`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminSiteConfigController.java#getConfig`
* **鉴权要求**：需登录；需角色：管理员。
* **功能说明**：返回后台当前可编辑的站点基础配置，用于初始化管理表单。
* **请求参数**：无。
* **响应字段**：
    * `id`：long，站点配置主键。
    * `version`：int，乐观锁版本。
    * `siteTitle`：string，网站标题。
    * `seoKeywords`：string，SEO 关键字。
    * `seoDescription`：string，SEO 描述。
    * `brandSlogan`：string，全局品牌主张。
    * `brandTagline`：string，辅助宣传语。
    * `logoLightMediaId`：long，亮色 Logo 资源 ID，可为空。
    * `logoLightUrl`：string，亮色 Logo 访问地址，可为空。
    * `logoDarkMediaId`：long，暗色 Logo 资源 ID，可为空。
    * `logoDarkUrl`：string，暗色 Logo 访问地址，可为空。
    * `updatedAt`：string，最近更新时间。
* **错误码**：
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `10002`：默认站点配置记录不存在。
* **备注**：站点配置为单例记录，接口只返回活跃配置。

### 更新站点基础配置

* **接口类型**：Admin
* **请求方法**：PUT
* **接口路径**：`/admin/api/site/config`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminSiteConfigController.java#updateConfig`
* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：更新网站 Title、SEO 信息、品牌文案及亮暗双 Logo。
* **请求参数**：
    * `version`：int，必填，当前配置版本号。
    * `siteTitle`：string，必填，最长 120 字符。
    * `seoKeywords`：string，选填，最长 255 字符。
    * `seoDescription`：string，选填，最长 500 字符。
    * `brandSlogan`：string，必填，最长 160 字符。
    * `brandTagline`：string，选填，最长 255 字符。
    * `logoLightMediaId`：long，选填，亮色 Logo 资源 ID。
    * `logoDarkMediaId`：long，选填，暗色 Logo 资源 ID。
* **响应字段**：同“获取站点基础配置”。
* **错误码**：
    * `10001`：请求参数缺失、长度不合法。
    * `10002`：站点配置或引用的媒体资源不存在。
    * `10003`：配置版本冲突，需要刷新后重试。
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `20005`：缺少 CSRF Token 或 Token 无效。
    * `30001`：Logo 引用的媒体不是可公开图片资源。
* **备注**：保存成功后写入审计日志，并在事务提交后失效 `official:portal:site-config` 缓存。

#### 1.2 导航菜单管理

### 获取导航菜单树

* **接口类型**：Admin
* **请求方法**：GET
* **接口路径**：`/admin/api/site/navigation/menus`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminNavigationMenuController.java#getMenuTree`
* **鉴权要求**：需登录；需角色：管理员。
* **功能说明**：返回后台可编辑的完整顶部导航树，包含隐藏菜单与二级菜单结构。
* **请求参数**：无。
* **响应字段**：
    * `id`：long，菜单 ID。
    * `parentId`：long，父菜单 ID，一级菜单固定为 `0`。
    * `menuLevel`：string，菜单层级，取值 `LEVEL_1` / `LEVEL_2`。
    * `menuName`：string，菜单名称。
    * `targetType`：string，跳转类型，取值 `GROUP` / `INTERNAL_ROUTE` / `PAGE_ANCHOR` / `EXTERNAL_LINK`。
    * `routePath`：string，内部路由，仅 `INTERNAL_ROUTE` 时返回。
    * `anchorCode`：string，页面锚点，仅 `PAGE_ANCHOR` 时返回。
    * `externalUrl`：string，外部链接，仅 `EXTERNAL_LINK` 时返回。
    * `openInNewTab`：boolean，外部链接是否新窗口打开。
    * `visible`：boolean，菜单是否显示。
    * `sortOrder`：int，当前排序值。
    * `version`：int，乐观锁版本。
    * `updatedAt`：string，最近更新时间。
    * `children`：array，二级菜单列表。
* **错误码**：
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
* **备注**：接口返回活跃菜单树，不返回逻辑删除记录。

### 新增导航菜单

* **接口类型**：Admin
* **请求方法**：POST
* **接口路径**：`/admin/api/site/navigation/menus`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminNavigationMenuController.java#createMenu`
* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：新增一级或二级导航菜单，并返回最新后台导航树。
* **请求参数**：
    * `parentId`：long，必填，父菜单 ID；一级菜单传 `0`。
    * `menuName`：string，必填，最长 64 字符。
    * `targetType`：string，必填，取值 `GROUP` / `INTERNAL_ROUTE` / `PAGE_ANCHOR` / `EXTERNAL_LINK`。
    * `routePath`：string，选填，内部路由。
    * `anchorCode`：string，选填，页面锚点，不带 `#`。
    * `externalUrl`：string，选填，外部链接，仅允许 `http/https`。
    * `openInNewTab`：boolean，必填，仅 `EXTERNAL_LINK` 时可为 `true`。
    * `visible`：boolean，必填，是否显示。
* **响应字段**：同“获取导航菜单树”。
* **错误码**：
    * `10001`：请求参数缺失、长度不合法。
    * `10002`：父菜单不存在。
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `20005`：缺少 CSRF Token 或 Token 无效。
    * `30002`：跳转目标配置不合法。
    * `30003`：菜单层级不合法。
    * `30004`：父菜单不存在或不可挂接。
    * `30005`：同级菜单名称重复。
* **备注**：新增菜单默认追加到同级末尾；二级菜单只能挂在一级菜单下。

### 更新导航菜单

* **接口类型**：Admin

* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminNavigationMenuController.java#updateMenu`
* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：更新菜单名称、跳转配置和显示状态，并返回最新后台导航树。
* **请求参数**：
    * `version`：int，必填，当前菜单版本号。
    * `menuName`：string，必填，最长 64 字符。
    * `targetType`：string，必填。
    * `routePath`：string，选填，内部路由。
    * `anchorCode`：string，选填，页面锚点。
    * `externalUrl`：string，选填，外部链接。
    * `openInNewTab`：boolean，必填。
    * `visible`：boolean，必填。
* **响应字段**：同“获取导航菜单树”。
* **错误码**：
    * `10001`：请求参数缺失、长度不合法。
    * `10002`：菜单不存在。
    * `10003`：版本冲突，需要刷新后重试。
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `20005`：缺少 CSRF Token 或 Token 无效。
    * `30002`：跳转目标配置不合法。
    * `30003`：菜单层级不合法。
    * `30004`：父菜单不存在或不可挂接。
    * `30005`：同级菜单名称重复。
* **备注**：不允许通过编辑修改菜单层级或二级菜单父级；存在子菜单的一级菜单只能配置为 `GROUP`。

### 更新导航菜单显示状态

* **接口类型**：Admin
* **请求方法**：PUT
* **接口路径**：`/admin/api/site/navigation/menus/{menuId}/visibility`

* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：单独切换菜单显示状态，并返回最新后台导航树。
* **请求参数**：
    * `version`：int，必填，当前菜单版本号。
    * `visible`：boolean，必填，是否显示。
* **响应字段**：同“获取导航菜单树”。
* **错误码**：
    * `10001`：请求参数缺失。
    * `10002`：菜单不存在。
    * `10003`：版本冲突，需要刷新后重试。
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `20005`：缺少 CSRF Token 或 Token 无效。

* **接口类型**：Admin
* **请求方法**：DELETE
* **接口路径**：`/admin/api/site/navigation/menus/{menuId}`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminNavigationMenuController.java#deleteMenu`

* **功能说明**：删除指定菜单，并返回最新后台导航树。
* **请求参数**：无。
* **响应字段**：同“获取导航菜单树”。

### 重排导航菜单

* **接口类型**：Admin
* **请求方法**：PUT
* **接口路径**：`/admin/api/site/navigation/menus/order`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminNavigationMenuController.java#reorderMenus`
* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：按同级范围提交完整菜单顺序，并返回最新后台导航树。
* **请求参数**：
    * `parentId`：long，必填，父菜单 ID；一级菜单排序传 `0`。

* **错误码**：
    * `10001`：请求参数缺失或列表为空。
    * `10002`：排序目标不存在。
    * `10003`：排序列表与当前同级菜单不一致。
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `20005`：缺少 CSRF Token 或 Token 无效。
* **备注**：只支持同级排序，不支持跨父级拖拽。

#### 2.1 首页首屏 Banner

### 获取首页首屏 Banner 配置

* **接口类型**：Admin
* **请求方法**：GET
* **接口路径**：`/admin/api/site/home-banner`

    * `text`：string，按钮文案，未启用时为空。
    * `targetType`：string，跳转类型，取值 `INTERNAL_ROUTE` / `PAGE_ANCHOR` / `EXTERNAL_LINK`。

    * `externalUrl`：string，外部链接，仅 `EXTERNAL_LINK` 时返回。
    * `openInNewTab`：boolean，外部链接是否新窗口打开。
* **错误码**：
    * `20001`：未登录访问。
    * `20002`：非管理员访问。
    * `10002`：默认 Banner 配置记录不存在。
* **备注**：Banner 配置为单例记录，接口只返回活跃配置。

### 更新首页首屏 Banner 配置

* **接口类型**：Admin
* **请求方法**：PUT
* **接口路径**：`/admin/api/site/home-banner`
* **实现位置**：`src/main/java/com/company/officialwebsite/modules/site/controller/AdminHomeBannerController.java#updateBanner`
* **鉴权要求**：需登录；需角色：管理员；必须携带有效 CSRF Token。
* **功能说明**：更新首页首屏主视觉文案、背景图和固定双 CTA 按钮配置。
* **请求参数**：
    * `version`：int，必填，当前配置版本号。

## 二、Portal 前台接口

> **接口组织说明**：Portal 接口按功能点顺序排列，与 Admin 接口对应：  
> 1. 全局配置（站点配置、导航菜单）

> 3. 企业实力模块（企业实力核心指标）

> 5. AI 战略模块（AI 战略卡片）  
> 6. 核心能力底座模块（分类与子项列表）  
> 7. 产品矩阵模块（产品矩阵列表、标杆案例列表）
