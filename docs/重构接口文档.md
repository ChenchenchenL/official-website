## Phase 1 基础设施说明

Phase 1 完成 `pagebuilder` 模块的数据库设计、实体类、枚举、Mapper 和 Schema 模型，**未涉及 Controller 接口**，故无 API 条目。

| 组件 | 说明 |
|------|------|
| 数据库迁移 | `V23__create_pagebuilder_tables.sql` — 6 张核心表 |
| 实体类 | `PageDefinitionEntity`, `PageDraftEntity`, `PageVersionEntity`, `PagePublishSnapshotEntity`, `ComponentTemplateEntity`, `PageDependencyEntity` |
| 枚举 | `PageStatusEnum`, `PageTypeEnum`, `ComponentTemplateStatusEnum`, `PageVersionSourceTypeEnum`, `PublishSnapshotStatusEnum`, `BindingTypeEnum`, `DependencyTypeEnum` |
| 模型 | `PageSchemaModel`, `LayoutModel`, `SectionModel`, `BindingModel`, `SeoModel` |
| 常量 | `PageBuilderConstants` — 缓存 Key 前缀 |

## 产品详情可视化编辑接口（S5）

产品编辑器固定使用以下专用接口，不得使用传统产品 CRUD 承载草稿、预览、发布、回滚或下线：

| 方法 | 路径 | Controller 方法 |
| --- | --- | --- |
| POST | `/admin/api/products/drafts` | `AdminProductEditorController#createDraft` |
| GET / PUT | `/admin/api/products/{id}/draft` | `getDraft` / `saveDraft` |
| POST | `/admin/api/products/{id}/draft/previews` | `createPreviewToken` |
| GET / DELETE | `/admin/api/products/{id}/previews/{previewToken}` | `getPreview` / `revokePreviewToken` |
| POST | `/admin/api/products/{id}/publish` | `publish` |
| GET | `/admin/api/products/{id}/versions` | `listVersions` |
| POST | `/admin/api/products/{id}/rollback/{versionId}` | `rollback` |
| POST | `/admin/api/products/{id}/offline` | `offline` |

预览创建必须提交当前 `draftHash`；访问时校验 Token 的资源、创建管理员与草稿哈希。保存、发布、回滚和下线必须携带锁 Token 与草稿 `version`。Portal 产品详情只读取最新已发布版本快照，写操作提交后失效产品列表及关联页面渲染、SEO 缓存。
